# 🛒 Ecommerce Hybrid Automation Framework

A professional-grade **Hybrid Framework** designed for automating E-commerce web applications using **Selenium WebDriver**, **TestNG**, and **Maven**. This project follows the **Page Object Model (POM)** design pattern for high maintainability and scalability.

## 🚀 Key Features
* **Design Pattern:** Optimized **Page Object Model (POM)** structure.
* **Reporting:** Beautiful and detailed **Extent Reports** with failure screenshots.
* **Logging:** Comprehensive tracking using **Log4j** logs.
* **Data-Driven:** External configuration management via `config.properties`.
* **Reliability:** Explicit and Implicit waits to handle dynamic web elements.
* **Browser Support:** Cross-browser testing capabilities (Chrome, Firefox, Edge).

## 🛠️ Technologies Used
* **Language:** Java (JDK 11+)
* **Automation Tool:** Selenium WebDriver (v4.x)
* **Test Runner:** TestNG
* **Build Tool:** Maven
* **Reporting Tool:** Extent Reports

## 📂 Project Structure
* `src/main/java`: Contains Page Objects and Utility classes.
* `src/test/java`: Contains Test Cases and Base Setup.
* `Configurations/`: Stores environment-specific properties.
* `Reports/`: Auto-generated HTML execution reports.
